# crow

An universal CLI tool used to Encrypt, Decrypt and Crack cyphers.

## Features
* **Automated Cracking:** Automatically tests all possible cryptographic shifts in milliseconds without needing the user to guess the key.
* **Chi-Squared Scoring:** Grades each possible decryption against standard English letter frequencies to mathematically identify the correct translation.
* **Instant Decryption Results:** Provides the exact Key and the resulting plaintext message directly in the terminal for immediate reading.
* **Non-Letter Preservation & Routing:** Intelligently bypasses numbers, spaces, and punctuation
